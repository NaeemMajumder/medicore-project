- button-export-report-pdf [Mandatory]
### User Story
As an admin, I want to see a complete system overview so that I can monitor hospital operations and performance at a glance.
### Acceptance Criteria
- Dashboard shows total patients, doctors, receptionists and today's appointments
- Monthly revenue and total appointments this month are displayed with comparison to last month
- Bar chart shows appointments per week
- Line chart shows revenue per month
- Admin can filter data by date range
- Export report as PDF button generates a full report
